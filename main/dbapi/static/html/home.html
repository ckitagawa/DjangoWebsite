Greetings Traveller

I'm an Engineering student, programmer, photographer, black-belt, gamer among much else. This website is a portfolio of the work I've done, pictures I've taken and thoughts I've had. I'm always up to meet new people and work on new projects as much as my degree will let me so if you want to get in touch you can contact me via email.

I wrote this website myself; the backend is written in Python - my favourite language - with Django REST and the frontend is a mix of jQuery and AngularJS with a healthy sprinkling of Bootstrap to save time. If you are interested in the design of the site you can fork it on GitHub.

If you have read this far I don't know what else I can say... Go poke around at the rest of the site!
